# What is a Duplicate Element?

A duplicate element is a value that appears more than once in an array.

# How it is works 
* Purpose
The program is used to find and display duplicate elements present in an array.
* Outer Loop
The outer loop selects one element at a time from the array for comparison.
* Inner Loop
The inner loop compares the selected element with all remaining elements in the array.
* Condition Check
If arr[i] == arr[j], it means the element appears more than once and is considered a duplicate element.
* Output
The program prints all duplicate elements found in the array using the brute-force approach with a time complexity of O(n²).
